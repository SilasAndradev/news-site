## 🚀 Instalando JIFB:

Para instalar o JIFB, siga estas etapas:

```
$ git clone https://github.com/SilasAndradev/JIFB.git
```

## ☕ Usando JIFB:

Para usar JIFB:

```
$ cd JIFB
$ pip install -r requirements.txt
$ python manage.py runserver
```
